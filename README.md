# PC-Connet
Connet your PC with Mobile using localhost 
## Before Downloading 
### Before downloading the program must allow to system to access and open any port 8080 to any device can access it. Run the following command to allow any devices access to your system : 
 ```netsh advfirewall firewall add rule name="Open Port 8080" dir=in action=allow protocol=TCP localport=8080```
 ```netsh advfirewall firewall set rule name="Open Port 8080" new profile=any edge=yes```
